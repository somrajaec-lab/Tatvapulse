# TatvaPulse

**Tatvacare Operations Command Center** — Internal action item tracker for the Ops team.

## Features
- Self-registration with username & password
- Create, edit, delete tasks with owner assignment
- Priority levels (High / Medium / Low) and statuses (Open / In Progress / Done / Blocked)
- Filter by status, owner, or priority
- One-click WhatsApp snapshot sharing
- Data persists in browser localStorage
